Penjelasan Metode

Metode Selisih Mundur

Metode selisih mundur menghitung turunan dengan rumus:

f'(x) ≈ [f(x) - f(x-h)] / h

Metode ini menggunakan titik saat ini (x) dan titik sebelumnya (x-h).

Metode Selisih Pusat

Metode selisih pusat menghitung turunan dengan rumus:

f'(x) ≈ [f(x+h) - f(x-h)] / (2h)

Metode ini menggunakan titik sebelum (x-h) dan setelah (x+h), memberikan akurasi yang lebih baik.

User Manual

  1. Masukkan fungsi JavaScript valid dalam input "Fungsi f(x)" (contoh: Math.pow(x,2) + Math.sin(x))
  2. Masukkan nilai x dimana turunan akan dihitung
  3. Masukkan nilai h (step size) - semakin kecil semakin akurat
  4. Klik tombol "Hitung Turunan"
  5. Lihat hasil perhitungan dan grafik fungsi
  6. Bandungkan hasil kedua metode
